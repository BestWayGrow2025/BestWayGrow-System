## Purpose

This document is the master index of the BestWayGrow project documentation.

It provides a single entry point to all technical documents in the repository.

Every documentation file has a specific purpose and should be updated only after verification.

---

## Documentation Rules

1. Documentation must always match the repository.
2. Never document assumptions.
3. Verify before writing.
4. One topic belongs to one document only.
5. Update documentation after verified repository changes.
6. Preserve the numbering sequence.
7. Do not duplicate information between documents.
