# BestWayGrow Repository Structure

---

## Purpose

This document defines the physical structure of the BestWayGrow repository.

Every repository file must belong to one logical family.

This document is the authoritative reference for repository organization.

---

## Repository Root

The repository contains source code, HTML pages, documentation, stylesheets, and supporting files.

All files must follow the BestWayGrow naming conventions.

---

## Repository File Families

### Core Layer

Prefix:

core_

Purpose:

Contains the enterprise framework responsible for booting, routing, initialization, security, orchestration, loading, validation, and shared platform services.
